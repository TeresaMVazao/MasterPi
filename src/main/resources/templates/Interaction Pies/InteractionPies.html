Interaction Pies

This is the page that allows the users to manage all the Interaction Pies that are present in the Museum. Here you can add new Interaction Pies to the system so that they can participate in the execution of the games they are prepared to execute as well as remove them from the system when such is required. To also check system health, users can use the Status Page to check the health of each Interaction Pi.

Add Interaction Pi

Here an administrator can choose to add a new Interaction Pi to the system so as to increase the number of machines that contribute to the execution of the several available games or that allow the museum to cover a new area with a new machine deployed for that purpose.
